# 🔄️ IDK FCA Auditorios
Web application developed to participate in the **Hackathon Celebrating 40 Years of the Information Technology Program** at the Faculty of Accounting and Administration (FCA - UNAM).

This project addresses the institutional need to manage the FCA’s auditoriums and event spaces, enabling efficient and centralized event management.

The system includes two main types of users:

* 👤 **Officials**: can create and track reservations, request specialized equipment, upload speaker information, and obtain confirmation documents for their approved events.
* 🛠️ **Administrators**: can manage reservations and users, review historical records, filter information, and export reports in PDF or XLS format.

With these features, `FCA Auditoriums` allows:

* ✅ Reducing venue management and approval times.
* ✅ Ensuring traceability and transparency in reservations.
* ✅ Improving communication between the departments involved in event organization.

The solution was built following a Minimum Viable Product (MVP) approach, in accordance with the phases and guidelines established by the hackathon.
